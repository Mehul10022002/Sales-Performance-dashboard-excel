## 📊 Sales & Business Performance Dashboard (Excel)

### 📌 Project Overview
This project focuses on analyzing sales, profit, and delivery performance across regions, product categories, and shipping modes using Excel.

### 🎯 Goal
To evaluate overall business performance and identify patterns affecting profitability and operational efficiency.

### 📈 Objectives
- Analyze sales and profit distribution across regions  
- Identify high-performing and low-performing product categories  
- Evaluate delivery performance across shipping modes  
- Understand trends in sales over time  

### 🛠 What I Did
- Built an interactive Excel dashboard using Pivot Tables and charts  
- Cleaned and structured raw data for analysis  
- Created visualizations for regional, product, and delivery insights  
- Derived key insights to support decision-making  

### 💡 Key Insights
- High sales regions do not always have high profit margins  
- Technology category drives the highest profitability  
- Some sub-categories generate high sales but low profits  
- Delivery performance varies significantly across shipping modes  

### ✅ Conclusion
The analysis highlights opportunities to improve pricing strategies, optimize product mix, and enhance delivery efficiency, enabling better data-driven decision-making.

### 🛠 Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboarding)
